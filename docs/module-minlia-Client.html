Class: Client

minlia~Client(config)

The main search client for MinLia. Provides full-text search capabilities with typo tolerance and custom ranking.

Example
const client = new MinLia.SearchClient({
  docSelector: ".product-card",
  searchableAttributes: ["title", "description"],
  customRanking: [{ attribute: "popularity", direction: 1 }]
});
client.init();
const results = client.apiSearch("laptop");

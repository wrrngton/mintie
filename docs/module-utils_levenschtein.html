(static) getLevenshteinDistance(a, b) → {number}

Calculates the Levenshtein distance between two strings. The Levenshtein distance is the minimum number of single-character edits (insertions, deletions, or substitutions) required to change one string into another.

Parameters:
Name Type Description
a string

The first string.
b string

The second string.

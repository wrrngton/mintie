Methods

(static) normalise(instance, term, type) → {Array.<string>}

Normalizes and tokenizes text for indexing or searching. For search queries, splits on whitespace. For documents, creates progressive token prefixes for prefix matching.

Parameters:
Name Type Description
instance Object

The MinLia client instance (unused but passed for consistency).
term string

The text to normalize.
type string

The normalization type: "search" for queries, "docs" for documents.
Source:

Returns:

Array of normalized tokens.

Type
Array.<string>
Example
// Search normalization
normalise(instance, "Hello World!", "search");
// Returns: ["hello", "world"]

// Document normalization (creates prefixes)
normalise(instance, "cat", "docs");
// Returns: ["c", "ca", "cat"]
